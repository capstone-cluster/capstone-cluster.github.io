Project CLUSTER

Multi-Purpose HPC Computing for the ECE Department

About

Our goal for this project is to design and build a cloud computing asset for UD's ECE Department

One part is a Kubernetes cluster to host automatically deployed web apps for classes and research

We will use Rancher to manage this cluster, and host a website where students can deploy these apps

Another part is a small scale version of UD's research supercomputers to learn and test new ideas

This will use the same software stack as the full scale to lower the barrier to entry for supercomputing
