Proxmox ESXI Comparison

December 2, 2019 by Daniel May

Proxmox Advantages:
Open-Source
Supports KVM
Better documented API

ESXI Advantages:
Commercial Support
Proven/Wellknown

Proxmox overall seems better for our use case. This is due to the fact that the API is easier/better docuented. Additinally the support for KVM makes it a clear winner.
